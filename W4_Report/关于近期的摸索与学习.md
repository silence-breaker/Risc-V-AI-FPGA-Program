#### `IcraftMdzThirdParty.deb`的作用
- **核心功能**：该包整合了模型运行所必需的**非复旦微自研**的第三方库。
- **具体内容**：
    - **OpenCV**：用于图像和视频的读取、预处理（缩放、归一化）以及后处理结果的绘制（画框、写字）。   
    - **FFmpeg**：提供多媒体编解码支持。    
    - **基础数学库**：支持各种数值计算的底层加速。

### Nanobot
#### 是什么
轻量化的openclaw
它的诞生就是为了抗议 OpenClaw 的过度臃肿。Nanobot 证明了 99% 的 OpenClaw 功能其实可以用 1% 的代码量实现。因为代码极短，你可以**一眼看透** AI 是如何处理你的数据的，安全性极高。如果你想自己改写 AI 的决策逻辑，Nanobot 只需要改几个文件；而在 OpenClaw 里，你可能会迷失在成千上万的文件目录中。

### Telegram+nanobot
#### 国外的手机号注册的WhatsApp在国内是否可以正常使用
将国外手机号改成国内手机号即可，没遇到注册不了的问题
#### telegram与WhatsApp的使用区别（便利性）
telegram大陆账号注册需要付43美元
尝试虚拟账号：生成虚拟账号的网站永久停运了
虚拟号码网站更新：

| **特性**    | **WhatsApp**        | **Telegram**       |
| --------- | ------------------- | ------------------ |
| **便利性核心** | **熟人沟通极快**，像发短信一样简单 | **多设备切换流畅**，功能极其丰富 |
| **多设备支持** | 依赖手机主账号，体验一般        | 账号完全云端，多端独立同步      |
| **文件传输**  | 会占手机空间，易过期          | 不占空间，永久云端存储        |
| **加人方式**  | 必须有手机号              | 可以只用用户名 (ID)       |
| **扩展功能**  | 基本没有                | 机器人、小程序、大型社群       |

优先选择云端的telegram，WhatsApp在telegram无法使用时进行使用
国内账户选择WhatsApp就是最好的选择
